CSS Grid

What is CSS grid?

CSS Grid helps organize content on a web page much like Flexbox.

Initialization

Before we can use CSS Grid, we need to declare a grid container.

Here's an example:

display: grid;

Columns

Content can be organized into columns.

Here's an example:

grid-template-columns: 50px 100px 50px;

The number of width declarations determine how many columns there will be.

Rows

Content can be organized into rows.

Here's an example:

grid-template-rows: 50px 100px 50px;

Similar to columns, the number of width declarations determine how many rows there will be.

Changing Size

There are multiple ways to declare the width of a row.

Here's an example:

grid-template-columns: auto 50px 10% 2fr 1fr;

Mnemonics

Having trouble remembering the difference between align and justify?

  • justify-content: longer word: horizontal alignment
  • align-items: shorter word: vertical alignment
