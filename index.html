CSS Grid

What is CSS grid?

CSS Grid helps organize content on a web page much like Flexbox.

Align

Items can be arranged within their cells to position toward the start, center, or end respectively.

Align means vertical positioning.

Individual Items

align-self:

start;

center;

end;

All Items

Mnemonics

Having trouble remembering the difference between align and justify?

  • justify-content: longer word: horizontal alignment
  • align-items: shorter word: vertical alignment

Areas

Sections of a grid can be combined into areas.

You fist need to declare a template to define

grid-template-areas:

"header header header"

". content ."

"footer footer footer";

But what if there's isn't a template already created?

A template can be created quickly on the fly with the below example:

Repeat Function

Let's say you have a grid that's 100x100.

It would be very tedious to declare every area over and over. So developers use a repetition function.

grid-template-columns:

repeat(10, 1fr);

In the above example, the column repeats four times each with a width of one fraction of the total.

Here's what the statement would look like without the function:

grid-template-columns:

1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr;

The function can even copy complex declarations:

grid-template-columns:

repeat(2,10px 50px 20px);

Min Max Function

Sometimes you want to define bounds to the resizing of elements within your grid.

The Problem

For example, if a window becomes too small or too large, some elements may extend beyond the edges of the container to look something like this:

grid-template-columns:

repeat(3, 150px);

1
2
3
4
5
6

The Solution

With minmax(), the rows or columns can stop resizing at specified measurements:

grid-template-columns:

repeat(3, minmax(10px, 150px));

1
2
3
4
5
6

In the above example, the columns can resize anywhere from 10px to 150px in width.

Try resizing the window to see it in action.

Auto-fill

Auto fill can dynamically adjust how many items populate based on the remaining room in a row or column.

Here's an example:

grid-template-columns:

repeat(auto-fill, minmax(75px, 1fr);

Auto fit is almost identical to auto fill, except it also stretches the items once there's ample space remaining.

Media Query

Sometimes you want to dynamically move areas of the grid around like you would with CSS motion.

A media query can do just that:

@media (max-width: 500px) {

"phone layout stylings" }

header
ad #1
content
ad #2
footer

Try resizing the window to see the layout change!
