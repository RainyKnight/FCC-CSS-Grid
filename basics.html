CSS Grid Basics

Initialization

Before we can use CSS Grid, we need to declare a grid container.

Here's an example:

display:

grid;

Columns

Content can be organized into columns.

Here's an example:

grid-template-columns:

1fr 2fr 1fr;

The number of width declarations determine how many columns there will be.

Rows

Content can be organized into rows.

Here's an example:

grid-template-rows:

50px 100px 50px;

1
2
3

Similar to columns, the number of width declarations determine how many rows there will be.
