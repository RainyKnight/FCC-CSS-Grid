Align

Items can be arranged within their cells to position toward the start, center, or end respectively.

Align means vertical positioning.

Individual Items

align-self:

start;

center;

end;

3

The self property defines that this CSS it applied to each individual item.

All Items

align-items:

center;

The items property defines that this CSS it applied to the container and affects all items within.
